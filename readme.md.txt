# Hielo y Aventura — Dashboard Financiero

**Análisis económico-financiero de la concesión turística**  
Glaciar Perito Moreno · Licitación N° 15 · APN  

## Acceso
🔗 [Ver dashboard](https://tuusuario.github.io/hya-dashboard)

## Instalación como app
- **Android**: Chrome → menú ⋮ → "Añadir a pantalla de inicio"
- **iOS**: Safari → compartir → "Añadir a pantalla de inicio"
- **Desktop**: Chrome → ícono ⊕ en la barra de direcciones

## Funcionalidades
- VAN · TIRM · Payback · ROI interactivos
- Sliders: WACC, tasas TIRM, estructura de costos
- 5 vistas: Parámetros · Resumen · Flujos · Sensibilidad · Tabla
- Matrices de sensibilidad VAN × FC y TIRM × tasas
- Análisis de 3 escenarios (pesimista / base / optimista)
- Tabla detallada por trimestre (48 períodos)
- Funciona offline (PWA)

## Stack
HTML · CSS · JavaScript · Chart.js 4.4 · IBM Plex fonts
